Este proyecto forma parte del Máster y consiste en resolver una serie de ejercicios del manejo de estructuras básicas, funciones, clases, excepciones y programación funcional (map, filter, reduce).

Condicionales e iteraciones  
Funciones  
Manejo de excepciones  
Programación orientada a objetos  
Uso de `map()`, `filter()` y `reduce()`  
Buenas prácticas y comentarios
